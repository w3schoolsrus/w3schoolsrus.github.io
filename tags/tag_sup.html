Определение и использование

Тег <sup> определяет верхний индекс. Надстрочный текст появляется на пол символа над обычной строкой и иногда отображается более мелким шрифтом. Надстрочный текст может использоваться для сносок, таких как WWW[1].

Совет: Используйте тег <sub> для определения подстрочного текста.

Настройки CSS по умолчанию

Большинство браузеров будут отображать элемент <sup> со следующими значениями по умолчанию:

Пример

sup {
  vertical-align: super;
  font-size: smaller;
}
