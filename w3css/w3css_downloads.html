Скачать W3.CSS

Скачать W3.CSS можно с официального сайта по этой ссылке: https://www.w3schools.com/w3css/4/w3.css.

Или с нашего сайта по ссылке: https://w3schoolsrus.github.io/lib/w3.css.

W3.CSS является бесплатным

W3.CSS является бесплатным к использованию. Лицензия не требуется.

Как использовать W3.CSS

Чтобы использовать W3.CSS, просто добавьте ссылку на "w3.css" в html-коде ваших веб-страниц:

Пример

<!DOCTYPE html>
<html>
<title>Моя веб-страница</title>
<meta name="viewport" content="width=device-width, initial-scale=1">

<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">
Попробуйте сами »

Или скачайте w3.css и запускайте данный фреймворк с вашего собственного веб-сайта:

Пример

<!DOCTYPE html>
<html>
<title>Моя веб-страница</title>
<meta name="viewport" content="width=device-width, initial-scale=1">

<link rel="stylesheet" href="w3.css">

Если вы поместили w3.css в подпапку, добавьте имя папки между слешами:

Пример

<!DOCTYPE html>
<html>
<title>Моя веб-страница</title>
<meta name="viewport" content="width=device-width, initial-scale=1">

<link rel="stylesheet" href="/имя_папки/w3.css">

Если вы поместите w3.css в корень вашего сайта, добавьте слэш:

Пример

<!DOCTYPE html>
<html>
<title>Моя веб-страница</title>
<meta name="viewport" content="width=device-width, initial-scale=1">

<link rel="stylesheet" href="/w3.css">

Проверка (валидация) W3.CSS

Проверка W3.CSS может вернуть некоторые ошибки.

Вы часто будете получать некоторые ошибки при запуске проверок W3C.
За пределами текущей спецификации W3C всегда будут некоторые специфические для браузера значения.

СвойствоОписание
pointer-events Недопустимое свойство в CSS3, но предлагается для CSS4.
Используется в W3.CSS для удаления указателя руки с отключенных элементов.
user-select Недопустимое свойство в CSS3, но предлагается для CSS4.
Используется в W3.CSS для удаления выделения текста из интерактивных элементов.
pseudo-class :valid Недопустимое значение в CSS3, но рекомендуется для CSS4.
Используется в W3.CSS для указания правильного ввода.
@-webkit-keyframes Добавлен в W3.CSS, чтобы анимации работали в браузерах webkit (более старые версии Chrome, Safari и Opera).

Настройка W3.CSS

W3.CSS включает в себя основные шрифты и размеры шрифтов.

Вы можете настроить W3.CSS для себя, изменив эти настройки:

Пример

html {
  font-family: Verdana, sans-serif;
  font-size: 15px;
  line-height: 1.5;
  font-weight: normal;
}

h1, h2, h3, h4, h5, h6 {
  font-family: "Segoe UI", Verdana, sans-serif;
  font-weight: 400;
  line-height: 1;
  margin: 20px 0;
}

h1 a, h2 a, h3 a, h4 a, h5 a, h6 a {font-weight: inherit;}
h1 {font-size: 36px}
h2 {font-size: 30px}
h3 {font-size: 24px}
h4 {font-size: 20px}
h5 {font-size: 18px}
h6 {font-size: 16px}

hr {
  height: 0;
  border: 0;
  border-top: 1px solid #eee;
  margin: 20px 0;
}

W3.CSS Нормализация

Нормализация таблицы стилей делается для того, чтобы:

  • Исправить ошибки браузера
  • Удалить несоответствия браузера
  • Обеспечить общие настройки браузера по умолчанию
  • Нормализовать стили всех элементов HTML
  • Добавить улучшения CSS

В W3.CSS используется извлечение из Normalize.css.

Это позволяет W3.CSS отображать HTML последовательно во всех браузерах и соответствовать современным веб-стандартам.
