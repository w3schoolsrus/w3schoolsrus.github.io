Вывод HEX-кода 216 веб-цветов на веб-странице с помощью JavaScript

Данный скрипт реализует возможность вывода HEX-кода выбранного цвета из 216 цветов стандартной палитры веб-цветов. При наведении мыши на определённый цвет можно увидеть HEX-кодировку цвета. Если щелкнуть по конкретному цвету, число зафиксируется. Для повторного возобновления работы скрипта просто перезагрузите веб-страницу Ctrl+R или кликните повторно по той ячейке, код цвета которой отображен на экране.

Для желающих узнать код программы можете посмотреть его с помощью нажатия на данной веб-странице правой кнопки мыши и выбрав в меню "Программный код страницы/Просмотр кода страницы" (или сочетание клавиш CTRL + U). Данный код можно скопировать и отредактировать в редакторе кода, подставляя любые цвета в любом количестве. Если Вам нужен результат, просто сохраните эту страницу локально.

А вот еще один маленький скрипт. Код маленький, таблиц много. Можно задавать их количество и цвета, с которых они стартуют. Приводим пример скрипта и его работы.

Пример скрипта

<script>
clr=new Array('00','20','40','60','80','a0','c0','ff'); // задать стартовые цвета
for (i=0;i<8;i++) // 8 - количество таблиц, должно соответствовать
{document.write("<table border=0 cellpadding=8 cellspacing=2>");
for (j=0;j<8;j++)
{document.write("<tr>");
for (k=0;k<8;k++)
{document.write('<td bgcolor="#'+clr[i]+clr[j]+clr[k]+'">');
document.write('<tt><font color="#'+clr[7-i]+clr[7-j]+clr[7-k]+'"> ');
document.write(clr[i]+clr[j]+clr[k]+' </font></tt></td>');}
document.write("</tr>");}
document.write("</table><br>");}
</script>
