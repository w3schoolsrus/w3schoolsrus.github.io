HTML

Язык для создания веб-страниц

HTML - язык гипертекстовой разметки, предназначенный для создания веб-страниц. HTML не является языком программирования. С помощью html-тегов проводится разметка на веб-странице, с помощью которой размещаются различные объекты веб-документа: заголовки, текст, изображения, таблицы, списки, фреймы, видеоплееры и т.д.

<body>

</body>
</html>

CSS

Язык для стилизации веб-страниц

CSS (каскадные таблицы стилей) - язык, который отвечает за визуальное представление информации и оформление веб-страницы. С помощью стилей проводится форматирования текста и другая стилизация (оформление) объектов на веб-странице. CSS обычно используется в паре с HTML.

JavaScript

Язык для программирования веб-страниц

JavaScript (JS) - язык программирования, третья составляющая (вместе с HTML и CSS) стандартных веб-технологий, применяется для создания веб-сайтов. JavaScript превращает статические веб-страницы в динамические, изменчивые и взаимодействующие с пользователем. Язык программирования JavaScript управляет мультимедиа, анимирует изображения, меняет текст на веб-странице и многое другое.

Python

Популярный язык программирования

Python (Пайтон) - высокоуровневый язык программирования общего назначения, ориентированный на повышение производительности разработчика и чтения кода. Применяется для программирования как общих приложений, так и в веб-программировании (создание сайтов). Рекомендуется для изучения начинающими, изучающих веб-программирование, а также как первый язык программирования для новичков благодаря своей простоте и доступности.

PHP

Язык программирования веб-сервера

PHP (препроцессор гипертекста) - скриптовый язык общего назначения, применяется для разработки веб-приложений. Поддерживается абсолютным большинством хостинг-провайдеров и является одним из основных языков, которые применяются для создания динамических веб-сайтов. С помощью PHP часто создают CMS - систему управления контентом (движок сайта). PHP работает на стороне сервера (backend), обеспечивая взаимодействие с пользователем. На PHP созданы такие сайты, как Facebook, Yahoo, Вконтакте, Википедия, движок Wordpress и др. Язык программирования PHP вы можете Выучить онлайн на сайте w3schools на русском.

Выучить PHP

jQuery

Библиотека JavaScript для разработки веб-страниц

jQuery - популярная JavaScript библиотека с открытым кодом. Синтаксис jQuery разработан, чтобы сделать ориентирование в навигации удобным благодаря выбору элементов DOM, создание анимации, обработки событий и разработке AJAX-приложений. Помогает созданию мощных динамических веб-страниц. Пройти полный курс онлайн по библиотеке jQuery вы можете на сайте w3schools на русском.

Выучить JQUERY
