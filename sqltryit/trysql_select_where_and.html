W3C WebSQL

WebSQL хранит базу данных локально на компьютере пользователя. Каждый пользователь получает свой собственный объект базы данных.

WebSQL Поддержка браузерами

WebSQL поддерживается в Chrome, Safari, Opera и Edge.(79).

Если вы используете другой браузер, вы все равно сможете использовать наш редактор пробной версии SQL, но с другой версией, с использованием серверного приложения ASP, с базой данных доступа только для чтения, где пользователям не разрешено вносить какие-либо изменения в данные.
